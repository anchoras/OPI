# OPI
Основы программной инженерии

## Natural Number Calculator

Приложение для проверки владения таблицей умножения.

### Логика работы
1. Введите три натуральных числа (положительные целые числа)
2. При нажатии на кнопку "Calculate":
   - Вычисляется произведение первого и второго числа
   - Результат сравнивается с третьим числом
   - Если произведение равно третьему числу, выводится "Верно", иначе "Неверно" + правильный результат.
