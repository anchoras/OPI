## Natural Number Calculator

Простое JavaFX приложение для сравнения произведения двух чисел с третьим числом.

### Описание
Приложение содержит:
- Три поля ввода для натуральных чисел
- Одно поле вывода для результата
- Кнопку "Calculate", которая активируется только когда все поля заполнены

### Логика работы
1. Введите три натуральных числа (положительные целые числа)
2. При нажатии на кнопку "Calculate":
   - Вычисляется произведение первого и второго числа
   - Результат сравнивается с третьим числом
   - Если произведение равно третьему числу, выводится 1, иначе 0

### Требования
- Java 11 или выше
- Maven 3.6+

### Запуск приложения

#### С использованием Maven:
```bash
mvn clean javafx:run
```

#### Или скомпилировать и запустить вручную:
```bash
mvn clean compile
java --module-path <path-to-javafx-lib> --add-modules javafx.controls -cp target/classes com.example.CalculatorApp
```
